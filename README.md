#. Project Title
** Aseem Farms Production and Information Management System

#. Problem Statement
** Managing farm records manually can be time consuming and prone to errors.Farms need a simple system to store, update and retrieve farm information effectively.

#. Objective
** To develop a system that helps manage farm production records while also providing useful external information such as weather conditions and crop reference data.

#. Key Features
- Generate realistic farm production records.
- Store farm records in an SQL Database.
- Add create, read, update and delete operations.
- Retrieve current weather information using an API.
- Scrape crop reference information from wikipedia.
- Export farm records to a CSV.
- Generate weather information using the open meteo API.

#. Outcome
The system will provide an efficient way to manage farm production recorsd, store data in a database retrieve useful agricultural information and export records for reporting.

#. Technologies
- Python
- SQL
- Requests
- BeautifulSoup
- CSV
- JSON
- Streamlit
